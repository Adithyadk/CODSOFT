## Overview

The Simple Calculator program is a basic console-based calculator that allows you to perform common arithmetic operations on two numbers. It provides an easy-to-use menu-driven interface and keeps track of your operation history for reference.

## Explanation of the Code

The program code consists of the following key elements:
- Inclusion of necessary header files for input/output, string manipulation, and mathematical operations.
- The `main()` function serves as the program's entry point and handles user interaction.
- The program displays a menu with options for calculation, viewing history, and exiting.
- Users can input two numbers and choose an operation, and the program displays the result or an error message.
- The program maintains an operation history for reference.

## How to Use the Calculator

### Start Using the Calculator

- Run the program, and you'll see the program's title and menu options.
- Choose an option by typing the corresponding number and pressing Enter.

### Perform a Calculation

- If you choose option 1 to calculate:
  - Input the first number and press Enter.
  - Input the second number and press Enter.
  - Choose an operation by typing `+`, `-`, `*`, `/`, or `%`.
  - The program displays the result of the calculation or an error message.

### View Operation History

- If you choose option 2 to view the operation history, the program displays a list of your previous calculations.

### Perform Additional Calculations

- After completing a calculation or viewing the history, you can choose to perform additional calculations or view the history again.

### Exit the Program

- If you choose option 3 to exit the program, it will terminate, and you'll return to your command prompt or terminal.

That's it! You can use the Simple Calculator program to perform basic arithmetic calculations and keep track of your calculation history.
