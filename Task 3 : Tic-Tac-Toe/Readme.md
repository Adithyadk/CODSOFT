# Tic-Tac-Toe Game

![Tic-Tac-Toe](tic_tac_toe.png)

This is a simple console-based Tic-Tac-Toe game that allows two players to play against each other.

## Game Overview

- Game Board: The game board is a 3x3 grid where players make their moves.

- Players: Two players participate, one assigned "X" and the other "O."

- Display Board: The current state of the board is shown in the console.

- Player Input: The current player is prompted to enter their move by selecting a cell on the board.

- Update Board: The game board is updated with the player's move, and it's displayed.

- Check for Win: The game checks if the current player has won after their move.

- Check for Draw: It determines if the game is a draw when there are no more available moves.

- Switch Players: The game alternates turns between "X" and "O" players after each move.

- Display Result: The game shows the result, whether it's a win for one of the players or a draw.

- Play Again: After each game, players can choose to play another round.


## Game Rules

- The game board is a 3x3 grid.
- Players are assigned "X" and "O."
- The game board is displayed with cell numbers to help players make their moves.
- Players take turns entering the cell number (1-9) where they want to place their symbol ('X' or 'O').
- The game updates the board with the player's move, checks for a win, and continues until one player wins or it's a draw.
- After each game, players can choose to play again or exit.

## How to Play

1. Run the program in your C++ development environment.

2. Start a new game.

3. Enter the cell number (1-9) where you want to place your symbol ('X' or 'O'). The grid is numbered from 1 to 9 from left to right, top to bottom.

4. Make a move. If the cell is empty, your symbol will be placed; otherwise, you'll be prompted to try again.

5. Continue playing. The game will alternate between players ('X' and 'O'). Keep making moves until one player wins or the game is a draw.

6. Game outcome. The game will display the result - either a win for one of the players or a draw.

7. Play again. After each game, the program will ask if you want to play again. If you choose 'y', a new game will start; if you choose 'n', the program will exit.

Enjoy playing Tic-Tac-Toe with a friend, and have fun!
