# To-Do List Manager

The To-Do List Manager is a console-based application that helps you organize and manage your tasks. With this program, you can add, mark tasks as completed, delete tasks, and view your to-do list efficiently.

## Table of Contents

- [Overview](#overview)
- [Features](#features)
- [How to Use](#how-to-use)
- [Getting Started](#getting-started)
- [License](#license)

## Overview

The To-Do List Manager is a simple, user-friendly tool for task management. It uses a linked list data structure to keep track of tasks and their completion status.

## Features

1. **Add Tasks**: You can add tasks to your to-do list, and each task is marked as pending by default.

2. **Mark Tasks as Completed**: When you complete a task, you can mark it as completed, and it will be reflected in the task list.

3. **Delete Tasks**: If a task is no longer relevant, you can delete it from your list.

4. **View Tasks**: You can view all your tasks, including their completion status (completed or pending), to stay organized.

## How to Use

1. **Run the Program**:
   - Compile and run the program using your C++ compiler.

2. **Menu Options**:
   - Choose an option from the menu by entering the corresponding number (1 to 5):
     - Option 1: Add a Task - Enter a task description when prompted.
     - Option 2: Mark a Task as Completed - Enter the index of the task you want to mark as completed.
     - Option 3: Delete a Task - Enter the index of the task you want to delete.
     - Option 4: View All Tasks - Displays all tasks in the list, including their status (completed or pending).
     - Option 5: Exit - Exits the program.

3. **Task Management**:
   - Add tasks to your to-do list.
   - Mark tasks as completed when they are finished.
   - Delete tasks that are no longer relevant.
   - View your to-do list to keep track of tasks and their completion status.

The program provides a simple and interactive way to manage your tasks. It helps you stay organized and ensures that you keep track of your pending and completed tasks.

Feel free to use and customize this program to suit your task management needs.

## Getting Started

1. Clone or download this repository to your local machine.

2. Compile and run the program using a C++ compiler. Ensure that you have a C++ development environment set up.

3. Follow the on-screen menu to add, mark as completed, delete, and view your tasks.

4. Enjoy the benefits of an organized and efficient task management system.
